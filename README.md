# 🚂 Take Your Seat

A real-time seat booking and allocation platform designed for managing industrial visits, trips, and group travel.

Each college can create its own isolated booking environment using a unique access code, allowing students to visually select seats while admins manage reservations and bookings.

---

## ✨ Features

- Real-time seat updates using Firebase Firestore  
- Visual bogie & seat layout  
- Multi-college (multi-tenant) architecture  
- One seat per user validation  
- Optimistic locking for conflict prevention  
- Admin dashboard for seat & booking management  
- CSV export for bookings  

---

## 🎨 Seat Status System

Seats are visually distinguished for clarity:

- 🟢 Available  
- 🔴 Booked  
- 🟡 Reserved  
- ⚪ Unavailable  
  
---

## 🛠 Tech Stack

- **Frontend:** React + Vite  
- **Styling:** TailwindCSS  
- **Database:** Firebase Firestore  
  

---

## 🎯 Use Case

Built for managing seat allocation in:

- Industrial Visits  
- Academic Trips  
- Group Travel  
- Event Seating  

---

## 🚀 Concept

Take Your Seat focuses on simplicity, visual clarity, and real-time consistency , replacing error-prone forms and manual seat coordination with an intuitive booking interface.

---

## ❤️ Inspiration

Originally created to solve real-world seat coordination challenges during a college industrial visit.
